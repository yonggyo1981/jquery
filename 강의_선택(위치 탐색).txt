탐색 선택
 1. 위치 탐색 선택자
	$("요소:first") ->요소 중 첫번째를 선택
	$("요소").first() -> 요소 중 첫번째 선택
	$("요소:first-of-type") 
	$("요소:first-child")
	
	$("요소:last") -> 요소 중 마지막을 선택
	$("요소").last() -> 요소 중 마지막을 선택
	$("요소:last-of-type")
	$("요소:last-child")
	
	// 홀수 번째 인덱스
	$("요소:odd")
	
	// 짝수 번째 인덱스 
	$("요소:even")
	
	// 특성 순서 - nth-child 
	$("요소:nth-child(숫자, 수식)")
	
	
	// 자식이 1개만 있는 선택 
	$("요소:only-child")
