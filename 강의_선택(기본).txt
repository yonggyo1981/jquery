기본 선택자
	(CSS 선택자 방식)
	선택방법 - jQuery 생성자 인수  - $("선택자")
	
	1. 직접 선택자 
		아이디로 선택
			#아이디명  -> $("#아이디명")
			
		클래스로 선택
			.클래스명 -> $(".클래스명")
		
		태그명으로 선택
			태그명 -> $("태그명")
			* ->전체 선택자 -> 모든 태그 선택 $("*")
		
		그룹선택자 
			콤마(,)로 구분해서 여러 선택자를 동일하게 적용
			
		종속 선택자 
			하나의 요소 하위의 속성으로 접근할때 
			p#txt -> p 태그에 id가 txt인 요소 
		
	2. 인접 관계 선택자
